# 从 0.7 升级至 0.9

**请务必为 TFG 创建一个新的游戏实例，并将您的存档复制到新实例中，而不要直接在原实例内升级！** 这样做更为安全，同时也便于您预览升级后的世界，以便在需要时回退并做好更充分的准备。

从 0.7 版本升级至 0.9 的存档大体上是安全的，但有几项事项您需要了解：

1.  格雷科技移除了其所有的动能机器。如果您正在使用动能输入仓、动能搅拌机、齿轮箱（指GregTech的，而非Create 的）或交流发电机，这些在升级时都会消失，因此您应先将它们在电弧炉或研磨机中回收。
2.  0.9 版本使用了格雷机械（Greate），这是一个为机械动力（Create）的机械组件添加了格雷科技（GregTech）等级体系的附属模组，因此您需要用等效版本替换所有的 Create 装置。所有旧的 Create 物品都可以通过放入合成栏中转换为它们对应的 Greate 等效物品。Greate 的组件有应力容量限制，因此您可能还需要重新布置任何大型的 Create 装置。GregTech 的交流发电机和齿轮箱可以用 Create: Additions 的交流发电机和电动机替代。
3.  若干锭、板和粗矿已改为使用不同的物品，而非 GregTech 生成的原有物品。这些物品带有 "REPLACE ME" 纹理，您可以将它们放入合成方格中来转换为新版本。
4.  树液龙头已改为使用 ArborFirmaCraft 的系统。您应能将旧龙头放入合成方格中转换为新龙头，并在木桶（而非桶）中收集乳胶。关于树木何时可被采集的机制也已改变，请查阅野外指南以获取更多信息——现在有几种新的可采集树脂的树木，可能更适应您所在的气候。
5.  早期任务章节已被彻底重做，因此如果您在意"任务进度"的损失，可以通过开启作弊模式，然后点击任务屏幕右下角的铅笔图标，再对任务或章节右键点击 > 强制完成，来手动标记完成。
6.  单格岩浆不再被软管滑轮视为无限源——您现在需要 2000 格。不过，在 Y=-55 左右深度的大多数岩浆湖应该都足够大了！或者，大型锅炉现在效率也高得多，仅两个处理原木的焦炉就足以驱动一个电力高炉——如果使用煤炭则更多！最后，如果您已处于 HV 或更高阶段，可以尝试新的蓄水器，仅需两个源方块即可获得无限岩浆。

如果您正处于蒸汽时代中期，我们强烈建议在升级前先进入 LV 阶段，因为蒸汽时代的发展路径已发生显著变化。
