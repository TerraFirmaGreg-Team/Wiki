# 0.11 升级指南
**请务必为 TFG 创建一个新的游戏实例，并将您的存档复制到新实例中，而不要直接在原实例内升级！** 这样做更为安全，同时也便于您预览升级后的世界，以便在需要时回退并做好更充分的准备。

如果您是从 0.9 版本升级，请先按照 [从 0.9 升级至 0.10 的说明](/modern/zh_cn/upgrade-guides/from-0.9-to-0.10) 进行操作。

从 0.10 版本升级至 0.11 的存档通常是安全的，但仍存在几项小幅的破坏性变更：

1.  **EV发展路线进行了大幅重构**：
    - 几乎所有与铀/钚/氡相关的合成配方均已调整，现在需要在火星上通过核裂变才能完成。
    - 纳米 CPU 晶圆现在需要核裂变科技线末期的产物才能制造。
    - 钨的提炼流程现已更新，步骤更为复杂。

2.  **风车与水车现调整为 Greate 等级**，且水车需放置在河流中才能发挥全部效率。如果您此前是"正确"使用它们（例如，在蒸汽时代阶段将水车置于河流中），那么此项改动对您的影响应该不大。但如果您曾在地下大量放置人工水源来驱动水车，请尽快更换其他动力来源！为平衡此项调整，风车帆板的造价现已更为低廉，且烈焰人燃烧室在钢铁时代即可使用。

3.  **移除了可编程电路卡模组**。这将影响您所有使用该模组的 AE2 自动化设置，您应改为每种电路使用不同的机器进行生产（或升级至 IV 阶段的多方块结构）。我们希望能尽量减轻此项改动带来的不便，因此，如果您仅为某个特定配方中的冷门电路编号而专门配置一整套组装机，请向我们提交 [问题报告](https://github.com/TerraFirmaGreg-Team/Modpack-Modern/issues)，我们将协助调整相关编号。

4.  **移除了下界合金与末影合金**。您将能在火星上获取它们所有重要的产出物。如果您仍存储有下界或末影空气，请尽快进行蒸馏处理，因为相关配方也将很快移除。

5.  **移除了所有经过加工的钍制品**（如锭、棒等）。请先将它们全部研磨成粉！

6.  如果您在从 0.7 升级至 0.9 后仍存有旧的"REPLACE ME"物品，**它们将被直接删除**，请务必提前完成全部转换！

7.  **移除了生橡胶条**。
