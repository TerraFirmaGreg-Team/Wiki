# 0.9 → 0.10 升级指南

**请务必为 TFG 创建一个新的游戏实例，并将您的存档复制到新实例中，而不要直接在原实例内升级！** 这样做更为安全，同时也便于您预览升级后的世界，以便在需要时回退并做好更充分的准备。

如果您是从 0.7 版本升级，请先按照 [从 0.7 升级至 0.9 的说明](/modern/zh_cn/upgrade-guides/from-0.7-to-0.9) 进行操作。

从 0.9 版本升级至 0.10 的存档通常是安全的，但仍存在几项小幅的破坏性变更：

1. 冰箱已被更换为新的 GregTech 版本，它拥有不同等级、能够识别 EU/t 并支持自动化。 您可以将旧冰箱转换为新版本，但这会清空其内部物品。 请在更新前取出冰箱内的所有物品！
2. 几乎所有与食物自动化相关的配方都已移至新的食物处理机和食物烤炉。
3. 若干矿石的分布位置发生了变动，最显著的是赛特斯石英、铝土矿以及钛铁矿脉现在仅存在于月球上。 请查阅您的野外指南以获取更多信息。
4. 离心石英砂现在产出下界石英而非赛特斯石英。 如果您需要无限的赛特斯石英，请前往月球寻找。
5. 紫颂果现在只能在月球的电力温室中种植，而不再是通过辐照普通水果获得。
6. 通过软管滑轮获取的岩浆不再是无限的。 您仍可将其用作燃料，但前期阶段现在也有其他几种选择可供尝试：杂酚油（来自木头或煤炭）、油砂、或来自新增的向日葵与油菜籽植物的种子油（以及生物柴油）！
7. 几乎所有与 AE2 相关的配方都已更改，因此您需要更新所有的样板。
8. 机械动力：汽鸣铁道（Create: Steam 'n' Rails）模组目前处于支持 Create 6 的测试版本，因此您的一些列车/转向架可能会不可见或出现显示问题。
9. GregTech 进行了一次主版本更新，包含多项破坏性变更，请同时查阅其 [更新日志](https://github.com/GregTechCEu/GregTech-Modern/releases)。

请查看 [TFG 更新日志](https://github.com/TerraFirmaGreg-Team/Modpack-Modern/releases) 以获取所有其他更改的完整列表。
