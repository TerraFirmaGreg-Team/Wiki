# 0.10 → 0.11 更新指南

\*\* 請務必先創建新實例再把存檔複製到新實例，絕對不要直接更新原實例！ \*\*
這麼做不只更安全也方便在更新後預覽新世界，並為復原做足準備。

若從 0.9 更新請按照 [0.10 版更新說明](/modern/en_us/upgrade-guides/from-0.9-to-0.10) 進行操作

0.10 版更新到 0.11 版基本上安全無虞，但仍有小部分的破壞性更新：

1. EV 發展的重大重製：

- 關於鈾/鈽/氡等相關配方都被更新，現在必須在火星才能核裂變
- 納米 CPU 晶圓現在需要核裂變科技的後期產物才能製造
- 鎢現在有了新的(也更複雜的)提煉工藝。

2. 風車與水車調整為 Greate 級，水車需要放在河流裡才能起到全部效能。 若以前「正確地」進行操作，像是在蒸氣時代將水車放在河裡，那麼這項更新對您不會有太大的影響。 而習慣在基地大量放置人工水源驅動水車的您，最好盡快改用其他動力源。 為了幫助平衡，風帆現在更便宜，並且鋼鐵時代就能使用烈焰使者燃燒室。
3. 移除程式設計電路板模組。 這影響了所有 AE2 自動化設定，您應為每個電路使用不同機器(或升級到IV多方塊)。 我們盡可能地減輕改動帶來的不便，因此，若您需要冷門的電路來對應全套組裝機裡的某個配方，請[告知我們](https://github.com/TerraFirmaGreg-Team/Modpack-Modern/issues/2105) ，我們將協助調整相關編號。
4. 移除地獄合金與終界合金。 您可以在火星取得它們的重要產出物。 也請盡快蒸餾您的地獄空氣與終界空氣庫存，因為相關配方也將要被移除。
5. 移除所有釷加工製品(如碇、棒等)。 將它們都磨成粉！
6. 若還存有「Replace Me」的物品，請盡速完成轉換，它們將被直接刪除！
7. 移除生橡膠條
